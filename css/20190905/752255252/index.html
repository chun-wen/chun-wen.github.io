簡介：

Sass 是一個 CSS 預處理器，而它有兩種撰寫的風格
1.一個是 SASS
2.另一個就是 SCSS(較常使用這種寫法，原因是跟原本css寫法類似，較為直覺好懂)

Sass常見編譯方式主要有下面三種
1.軟體編譯 EX：prepros
2.gulp、webpack 前端任務/打包工具
3.編輯器內建的插件

接下來，就讓我介紹軟體編譯、Vscode內建的插件使用方式

今天將介紹sublime、Vscode兩種編輯器處理方式
