閉包

補充一下 Scope Chain 概念

  • 內層可以讀取外層變數，但外層不能讀取內層變數。當外層找不到變數時，會ㄧ直往外尋找直到 Global

子函式可以使用父函式變數，優點

  • 變數作用域切割是以 Function 為單位，因此變數不會污染 Global(範圍鏈概念)
  • 當持續被呼叫時，記憶體不會被釋放直到不使用

為何我們要使用到閉包？
